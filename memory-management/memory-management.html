How it works

MVT (Multiprogramming with Variable Number of Tasks)

Memory is divided into variable-sized partitions. Processes are allocated memory based on their exact size requirements, allowing efficient memory utilization. Supports dynamic allocation and deallocation.

MFT (Multiprogramming with Fixed Number of Tasks)

Memory is divided into fixed-sized partitions. Each process must fit within a partition, leading to potential internal fragmentation when process size doesn't match partition size.

External Fragmentation

Occurs when free memory is scattered in small non-contiguous blocks. Total free memory may be sufficient, but no single block is large enough to accommodate a new process.

Internal Fragmentation

Occurs when allocated memory is larger than required by the process. The unused space within the allocated partition is wasted and cannot be used by other processes.

Compaction

Process of moving all allocated memory blocks to one end to combine all free blocks into a single large block. Eliminates external fragmentation but requires system overhead.
